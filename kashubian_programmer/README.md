Kashubian Programmer keyboard
==============

To run
-----------
Navigate into kashubian_programmer repo
kmc build .
km-package-install -f build/kashubian_programmer.kmp

Description
-----------
Desktop: AltGr gives Polish letters (ą, ć, ę, ł, ń, ó, ś, ź, ż); Right Ctrl gives Kashubian-specific ã, é, ë, ò, ô, ù. Caps Lock+AltGr/Right Ctrl mirrors Shift for uppercase output.

Touch (phone/tablet): long-press the base keys to get the accented forms. Examples:  E → ę/Ę, ë/Ë; O → ò/Ò, ó/Ó, ô/Ô; Z → ż/Ż, ź/Ź; 0 → ò/Ò. Use Shift to access uppercase variants in the long-press menus.
