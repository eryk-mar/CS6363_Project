Kashubian Programmer keyboard
==============

Public Domain

Description
-----------
Kashubian Programmer layout. AltGr outputs: ã, é, ë, ò, ô, ù, ł, ż, plus Polish letters ą, ć, ę, ń, ó, ś, ź. Touch layouts expose the same characters via long-press on their base keys (ò on long-press 0). Caps Lock+AltGr works the same as Shift+AltGr for uppercase forms.
