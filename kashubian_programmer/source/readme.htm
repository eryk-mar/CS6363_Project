Kashubian Programmer

Desktop: AltGr outputs the Polish letters (ą, ć, ę, ł, ń, ó, ś, ź, ż); Right Ctrl outputs the Kashubian-specific ã, é, ë, ò, ô, ù. Caps Lock + AltGr/Right Ctrl works the same as Shift for uppercase.

Touch (phone/tablet): long-press the base keys to access the same letters (e.g., W → é/É; E → ę/Ę, ë/Ë; O → ò/Ò, ó/Ó, ô/Ô; Z → ż/Ż, ź/Ź; 0 → ò/Ò). Use Shift to see uppercase variants in the long-press menu.
