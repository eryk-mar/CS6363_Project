Start Using Kashubian Programmer

Kashubian Programmer follows a Programmer-style layout with AltGr for Kashubian letters and retains Polish letters on AltGr. Caps Lock+AltGr works the same as Shift+AltGr for uppercase forms.

Desktop

  • AltGr+Q = ã, AltGr+Shift+Q = Ã
  • AltGr+W = é, AltGr+Shift+W = É
  • AltGr+R = ë, AltGr+Shift+R = Ë
  • AltGr+P = ô, AltGr+Shift+P = Ô
  • AltGr+0 = ò, AltGr+Shift+0 = Ò
  • AltGr+U = ù, AltGr+Shift+U = Ù
  • AltGr+A/C/E/L/N/O/S/X/Z for Polish ą, ć, ę, ł, ń, ó, ś, ź, ż (AltGr+Shift+X/Z for Ź/Ż)

Touch

Long-press base keys for Kashubian/Polish characters (for example, q for ã, w for é, r for ë, p for ô, 0 for ò).
